## Overview

The **SOC Level 2** path focuses on advanced skills for a Tier 2 Security Operations Centre analyst. The primary emphasis is on **malware analysis** — understanding how malicious code works at the assembly and binary level, how to analyze PE files, and how to use both static and dynamic analysis techniques to reverse-engineer malware samples.

Key skill areas covered across this path:

- x86 architecture and assembly language fundamentals
- PE file structure and header analysis
- Static analysis: strings, imports, entropy, YARA rules
- Dynamic analysis: behavioral observation, API monitoring, debugging
- Anti-reverse engineering techniques used by malware authors
- Malicious document analysis
- Adversary emulation with CALDERA (MITRE ATT&CK)
